To make headings, you can use the h# element. There are different levels of headings from h1 which is the biggest to h6 which is the smallest. The heading you use signifies how important your heading is.

Bolding

As for learning how to bold, there are 2 ways you can do so. The first is using the b element. Like this: I am bold. Or you can use the strong element like this: I am strong.

Either way, you will likely use the strong element more that the bold element. Another thing to note is that you will be use bold in between other elements like this: Hello, my name is Strong


Italicizing

Just like bolding, there are also two elements that can help make text italicize. The first way is by using the em element like this: I am italicized!. Or you can use the i element like this: I am also italicized!

Either way, just like the bolding method, you will be using these elements with text. For example: Who are you?
